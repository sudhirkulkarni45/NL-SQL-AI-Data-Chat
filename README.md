NL‑SQL: AI Data Chat Assistant
📌 Overview
NL‑SQL is an AI Data Chat assistant that transforms natural language into SQL queries. It enables dynamic refinement, insight extraction, and visualization recommendation. Built with Python, Flask, and GCP, it features conditional execution and progressive rendering for optimized latency and user experience.

🚀 Features
Natural Language to SQL – Converts user queries into optimized SQL.

Dynamic Refinement – Iteratively improves results based on feedback.

Insight Extraction – Summarizes and highlights key findings.

Visualization Recommendation – Suggests charts/graphs tailored to context.

Progressive Rendering – Loader animations and staged outputs for better UX.

Conditional Execution – Reduces latency by executing only necessary steps.

Cloud Deployment – Runs reliably in GCP with resource optimization.

🛠️ Tech Stack
Languages: Python

Frameworks: Flask

Cloud: Google Cloud Platform (asia‑south1 region)

AI Models: Gemini, LangChain (for orchestration)
